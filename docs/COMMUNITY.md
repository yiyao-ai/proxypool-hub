# Community

CliGate maintains both repository-native support channels and direct community contact paths.

## Official Project Channels

### GitHub Discussions

Use Discussions for:

- product usage questions
- setup help
- roadmap and feature discussion
- community feedback

Link:

- `https://github.com/codeking-ai/cligate/discussions`

### GitHub Issues

Use Issues for:

- confirmed bugs
- release or packaging problems
- actionable documentation defects
- clearly scoped feature requests

Link:

- `https://github.com/codeking-ai/cligate/issues`

## Which Channel Should Users Choose

- Use `GitHub Issues` for bugs and release failures.
- Use `GitHub Discussions` for usage questions and open-ended discussion.
- Use `Discord` for community chat and quick follow-ups.
- Use `X` for release updates and external project visibility.
- Use `WeChat` for direct Chinese-language contact with the maintainer.

## Security Note

Do not send tokens, cookies, refresh credentials, or other secrets through public channels. For sensitive issues, follow [SECURITY.md](../SECURITY.md).
