What it is

One local gateway for coding tools and runtime workflows

CliGate connects Claude Code, Codex CLI, Gemini CLI, OpenClaw, dashboard chat, local runtimes, and channel workflows through one local control plane.

Default address

http://localhost:8081

Start the service, open the dashboard, add one usable credential, and then configure the client you want to proxy.
